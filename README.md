# claude-gtm

Claude Code плагин для разработки Go-To-Market стратегии через структурированный диалог.

## Что это

Плагин проводит тебя через все этапы создания GTM стратегии:

1. **Продукт** — что делаешь, какую проблему решаешь
2. **Аудитория** — кто покупает, сегменты, персоны
3. **Конкуренты** — кто на рынке, как отстроиться (с веб-поиском)
4. **Каналы** — где продвигаться, приоритизация
5. **Контент-стратегия** — о чём писать, в каком тоне
6. **Тактика** — контент-план на 12 недель

На выходе получаешь:
- `docs/gtm/strategy.md` — стратегический документ
- `docs/gtm/content-plan.md` — конкретный план действий

## Установка

```
/install-plugin github:inem/claude-gtm
```

## Использование

### Полный цикл

```
/gtm
```

Проведёт через все этапы последовательно. Можно остановиться и продолжить позже — прогресс сохраняется в `docs/gtm/context.yaml`.

### Отдельные этапы

Если нужен только конкретный этап:

```
/gtm:product      — анализ продукта
/gtm:audience     — целевая аудитория
/gtm:competitors  — анализ конкурентов
/gtm:channels     — каналы продвижения
/gtm:content      — контент-стратегия
/gtm:tactics      — тактический план
```

## Структура результатов

```
docs/gtm/
├── context.yaml        # Вся собранная информация (YAML)
├── strategy.md         # Итоговый стратегический документ
├── content-plan.md     # План на 12 недель
└── research/
    ├── competitors.md  # Детальный анализ конкурентов
    └── market.md       # Тренды и инсайты
```

## Особенности

- **Адаптивный диалог** — простые вопросы группируются, сложные задаются по одному
- **Веб-поиск** — автоматически ищет и анализирует конкурентов
- **Гибкость** — можно пропускать этапы, возвращаться, настраивать глубину
- **Универсальность** — работает для B2B, B2C, услуг, маркетплейсов
