# Каналы продвижения

## Цель этапа

Определить через какие каналы достучаться до аудитории. Приоритизировать по соотношению усилия/результат. Учесть ресурсы и стадию продукта.

## Входные данные

Прочитай `docs/gtm/context.yaml` — нужна информация о:
- Тип продукта и стадия (product)
- Где обитает аудитория (audience)
- Что делают конкуренты (competitors)

## Вопросы

### Ресурсы

1. Какой бюджет на маркетинг?
   - Нет бюджета (только время)
   - Небольшой ($100-500/мес)
   - Средний ($500-2000/мес)
   - Значительный ($2000+/мес)

2. Сколько времени можешь уделять маркетингу?
   - Пара часов в неделю
   - Несколько часов в день
   - Полный рабочий день
   - Есть команда

3. Какие навыки есть?
   - Писать тексты/контент
   - Дизайн
   - Видео
   - Техническое SEO
   - Paid реклама
   - Продажи/outreach

### Предпочтения

4. Какие каналы уже пробовал? Что работало/не работало?

5. Есть ли уже аудитория где-то? (блог, соцсети, email-лист, сообщество)

6. Готов ли к личному бренду? (выступать лицом продукта)

## Анализ каналов

На основе собранных данных оцени каналы:

### Для B2B

| Канал | Подходит если... | Усилия | Время до результата |
|-------|------------------|--------|---------------------|
| LinkedIn контент | Есть экспертиза, готов писать | Средние | 3-6 мес |
| LinkedIn outreach | Чёткий ICP, небольшой рынок | Низкие | 1-2 мес |
| SEO/Контент | Есть поисковый спрос | Высокие | 6-12 мес |
| Product Hunt | Tech аудитория, есть продукт | Средние | Сразу |
| Партнёрства | Есть смежные продукты | Средние | 2-4 мес |
| Cold email | Понятный ICP, B2B | Низкие | 1-2 мес |
| Вебинары | Есть экспертиза, сложный продукт | Высокие | 2-3 мес |
| Paid (Google/LinkedIn) | Есть бюджет, понятный intent | Низкие* | Сразу |

### Для B2C

| Канал | Подходит если... | Усилия | Время до результата |
|-------|------------------|--------|---------------------|
| TikTok/Reels | Визуальный продукт, молодая ЦА | Высокие | 1-3 мес |
| Instagram | Визуальный продукт, lifestyle | Средние | 3-6 мес |
| YouTube | Есть что показать, готов на видео | Высокие | 6-12 мес |
| SEO | Есть поисковый спрос | Высокие | 6-12 мес |
| Influencers | Есть бюджет, понятная ниша | Средние | 1-2 мес |
| Community | Есть активная ниша | Средние | 3-6 мес |
| Paid (Meta/Google) | Есть бюджет | Низкие* | Сразу |
| Viral/referral | Продукт имеет сетевой эффект | Высокие | Непредсказуемо |

*Усилия низкие, но нужен бюджет

## Выходные данные

Запиши в `docs/gtm/context.yaml` секцию `channels`:

```yaml
channels:
  resources:
    budget: "small"  # none | small | medium | large
    time: "few_hours_week"
    skills:
      - "writing"
      - "technical"
    existing_audience:
      - platform: "Twitter"
        size: 500

  primary:
    - name: "LinkedIn контент"
      priority: 1
      rationale: |
        Аудитория там, есть экспертиза писать,
        не требует бюджета.
      effort: "medium"
      time_to_results: "3-6 months"
      tactics:
        - "2-3 поста в неделю"
        - "Комментинг у лидеров мнений"

    - name: "Product Hunt launch"
      priority: 2
      rationale: |
        Tech аудитория, одноразовое усилие,
        можно получить первых пользователей.
      effort: "medium"
      time_to_results: "immediate"

  secondary:
    - name: "SEO/Blog"
      priority: 3
      rationale: |
        Долгосрочная инвестиция,
        начать параллельно с основными каналами.

  not_recommended:
    - name: "Paid ads"
      reason: "Нет бюджета на данной стадии"
    - name: "TikTok"
      reason: "B2B аудитория не там"

  experiments:
    - name: "Cold outreach"
      hypothesis: "Можно получить 5 демо из 100 писем"
      success_criteria: ">3% response rate"
```

## Критерий завершения

- Определены 2-3 приоритетных канала с обоснованием
- Учтены ресурсы (бюджет, время, навыки)
- Есть понимание что НЕ делать
- Пользователь согласен с приоритизацией
